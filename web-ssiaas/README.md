# Vertex Web SSIaaS

Vertex Web SSIaaS is an open-source Identity-as-a-Service platform designed to simplify the creation, issuance, and management of Self-Sovereign Identities (SSI) and Verifiable Credentials (VCs). 

The system abstracts the underlying complexities of cryptographic signing and decentralized networks. It allows issuers to focus on credential design and distribution through an intuitive, No-Code interface, while embedding cryptographic VCs directly into standard PDF files. This hybrid approach enables offline-ready verification and utilizes OIDC for secure user authentication.
